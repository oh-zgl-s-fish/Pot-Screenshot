### What to need

- Snipaste (free or pro)
- Pot-app

### What can it do

- Use [Snipaste](https://www.snipaste.com/) to get a screenshot and use [Pot](https://pot-app.com/) to translate the text with OCR
- This is more user-friendly for touch screen users
- So keep the above two programs running in the background is necessary

### How to Use

- Go to the [Releases](https://github.com/oh-zgl-s-fish/Pot-Snipaste/releases) to download the latest version or build it for yourself
- Unzip it and put files in a folder you will never deleta
- Edit the `config.ini` and change the path to sreenshot app and the port of Pot to the right one, more infomation look [here](https://github.com/oh-zgl-s-fish/Pot-Snipaste/wiki)
- Launch Snipaste-pot.exe, maybe the name will be changed.

### How to build

- Download `Bat to Exe Converter` made by Faith Kodak or other tools can convert bat to exe and the [AutoHotKey v1.1](https://www.autohotkey.com/)
- Convert the bat file to an exe file with the option `Windows (Invisible)`
- Convert the ahk file to an exe file

### 这个项目需要什么

- 截屏工具Snipaste
- Pot-app

### 这个项目能干什么

- 使用 [Snipaste](https://www.snipaste.com/) 截屏并使用 [Pot-app](https://pot-app.com/) 完成文本的OCR和翻译

- 所以这个项目需要上述的两个程序处在后台

### 如何使用

- 去 [Releases](https://github.com/oh-zgl-s-fish/Pot-Snipaste/releases) 界面下载最新版本或者自己构建

- 编辑 `config.ini` 并编辑截图工具Snipaste的正确路径和Pot的正确端口，更多信息请自行查阅[wiki](https://github.com/oh-zgl-s-fish/Pot-Snipaste/wiki)
- 启动 Snipaste-pot.exe，将来也许会改名

### 如何构建

- 下载 `Bat to Exe Converter` 作者是 Faith Kodak 或者其他可以转换bat到exe的工具和 [AutoHotKey v1.1](https://www.autohotkey.com/)
- 转换bat到exe并且选择有 `Windows (Invisible)` 的选项
- 转换ahk到exe
